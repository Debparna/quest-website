How it Works

Composable Smart Contracts, Tokens & NFTs to drive Hyper-personalized engagement and build User Loyalty in way that was never possible before.
Drop Badges
Acknowledge and Reward User contributions using Soul Bound Token(SBT) Badges which contain Loyalty Points.
Issue Membership Cards
Badges get added to a Membership Dynamic NFT that visually evolves as the member continues to contribute.
Gain Dynamic Loyalty Points
Members can gain Loyalty Points from Badges they collect, level up and unlock unique personalized rewards.

Quest Platform Features
